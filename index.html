Automated Server Provisioning and Configuring using Infrastructure as Code

Project Description

InProv is an automated infrastructure provisioning solution making use of DevOps and Infrastructure as Code practices, as well as various local and cloud technologies to provision and configure established Linux servers effortlessly.

An implementation of InProv in enterprise would allow infrastructure engineers and Linux system administrators to free themselves from the burden of manual provisioning.
